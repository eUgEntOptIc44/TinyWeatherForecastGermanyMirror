New in version ([VERSION]):

- wind speed can now be displayed in charts
- new determination of the next weather station in the background
- supports WeatherSpec v4, which allows to display much more weather data on some wearables via Gadgetbridge
- new dew point display in the weather details
- the number of affected locations in shared warnings can be limited
- minor bug fixes and further improvements

New in previous version:

- new: weather maps, pollen forecast, UV hazard index
- supports UV hazard index and precipitation on wearables with Gadgetbridge
- numerous, further improvements
